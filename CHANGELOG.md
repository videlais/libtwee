## [1.0.1] - 17 Oct 2025

### Fixed

- `Twine2HTML.Parse`: Fixed `KeyNotFoundException` by including empty 'position'/'size' attributes in passage Metadata when the attribute exists.

## [1.0.0] - 1 Feb 2025

### Added

- Initial release of libtwee.
- Support for parsing and compiling Twine 1 and Twine 2 HTML formats.
- Support for Twee source format.
- JSON export support.
  
This is the first public release of libtwee, a C# library for parsing and compiling current and historical Twine file formats.

This release can be found on NuGet for easier loading into projects.
